# Spring - Multi Kafka broker examples 

## how to run

### starting kafka infra 

```shell
# kafka 실행
make up

# kafka 종료
make down

# kafka 프로세스 확인
make ps 
```

### kafka ui 접속

```shell
http://localhost:9999
```
